# EzPanel

**语言**: [English](./README.md) | 简体中文 | [繁體中文](./README_TW.md) | [Русский](./README_RU.md) | [فارسی](./README_FA.md)

EzPanel 是一个功能完善的代理服务用户管理面板，支持 XrayM/XrayR 节点对接，提供完整的用户订阅、套餐管理、节点监控等功能。

## 版本说明

### Community（社区版）— 永久免费

社区版包含个人用户的基础功能，满足普通用户的正常使用。**永久免费，无使用期限。**

- 用户管理、订阅管理
- 节点管理（XrayR/XrayM 对接）
- 套餐管理
- 系统管理

> 社区版为闭源软件，官方不提供技术支持和定制需求。如需获取源代码，请购买商业授权。

### Pro（专业版）— 商业授权

在社区版基础上增加：

- 订单 / 发票 / 营销管理
- 支付网关集成（支付宝、微信、Stripe 等）
- 工单系统
- 知识库
- 佣金 / 推广系统
- 财务报表与数据分析
- **源代码仓库访问权限**（授权有效期内）

### Enterprise（企业版）— 商业授权

在专业版基础上增加：

- SSH 节点远程管理
- 应用内购（IAP）支持

> 购买 Pro 或 Enterprise 授权请访问 [https://opine.work](https://opine.work)。

## 多语言支持

EzPanel 内置国际化支持，开箱即用：

| 语言 | 代码 |
|------|------|
| 简体中文 | zh-CN |
| English（英语） | en |
| 繁體中文（繁体中文） | zh-TW |
| Русский（俄语） | ru |
| فارسی（波斯语） | fa |

用户可在界面右上角切换语言。切换后需在**管理后台 → 系统管理**中重置邮件模板和订阅模板，以使用对应语言版本。
