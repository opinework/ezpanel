# 用户手册

**语言**: [English](./04_user-guide.md) | 简体中文 | [繁體中文](./04_user-guide_TW.md) | [Русский](./04_user-guide_RU.md) | [فارسی](./04_user-guide_FA.md)

本文档面向 EzPanel 的普通用户，介绍注册、登录、订阅获取、套餐购买等功能的使用方法。

---

## 注册与登录

### 注册账号

1. 打开面板地址，点击「注册」
2. 填写邮箱和密码
3. 如果站点开启了邮箱验证，请查收邮件并点击验证链接
4. 如果站点要求邀请码，填写邀请码后提交

### 登录

输入注册时使用的邮箱和密码登录。勾选「记住我」可延长登录有效期。

### 忘记密码

1. 在登录页点击「忘记密码」
2. 输入注册邮箱，系统会发送重置邮件
3. 点击邮件中的链接设置新密码

---

## 两步验证（2FA）

开启两步验证可有效防止账号被盗。

### 开启 2FA

1. 进入「个人设置」→「安全设置」
2. 点击「开启两步验证」
3. 用 Google Authenticator 或 Aegis 等应用扫描二维码
4. 输入 App 中显示的 6 位验证码确认开启

### 登录时使用 2FA

登录时输入邮箱密码后，会要求输入 6 位动态验证码。

> **提示**：请务必保存好备用恢复码，丢失后无法找回。

## 订阅

订阅链接用于将节点信息导入到客户端（如 Clash、Sing-box 等）。

### 获取订阅链接

1. 登录后进入「我的订阅」或首页
2. 复制订阅链接
3. 粘贴到客户端的「添加订阅」处并更新

### 订阅格式

EzPanel 支持多种订阅格式，常见客户端对应格式：

| 客户端 | 推荐格式 |
|--------|----------|
| Clash / Mihomo | Clash 格式 |
| Sing-box | Sing-box 格式 |
| v2rayN / Nekoray | 通用 Base64 |
| Quantumult X | 通用格式 |

> 具体格式参数可能因站点配置而有所不同，请联系站点管理员确认。

### 订阅刷新

订阅内容会随节点变化而更新。建议在客户端中开启定时自动更新（如每小时或每天一次）。

## 套餐购买（Pro 版功能）

> 以下功能需要站点安装专业版授权后才可使用。

### 购买套餐

1. 进入「套餐」页面，浏览可用套餐
2. 选择套餐和计费周期（月付 / 季付 / 年付等）
3. 确认订单，选择支付方式
4. 完成支付后套餐立即生效

### 续费

套餐到期前可进入「我的订阅」提前续费，续费从当前到期时间顺延。

### 流量用完怎么办

流量耗尽后订阅将暂停，可选择：
- 购买流量包补充（如支持）
- 等待下个计费周期自动重置
- 手动续费触发流量重置

## 卡密兑换

部分站点支持通过卡密兑换套餐或余额。

1. 进入「卡密兑换」或「兑换」页面
2. 输入卡密代码
3. 点击「兑换」，成功后余额 / 套餐立即到账

---

## 邀请好友

1. 进入「邀请」或「推广」页面
2. 复制专属邀请链接或邀请码
3. 分享给好友，好友注册并消费后你会获得佣金

### 查看佣金

在「推广 / 邀请」页面可查看：
- 已邀请用户数
- 累计佣金金额
- 可提现余额

### 申请提现（Pro 版）

满足最低提现金额后，在「提现」页面发起申请，管理员审核后打款。

---

## 工单（客服）（Pro 版功能）

如遇问题需要联系站点客服：

1. 进入「工单」→「新建工单」
2. 选择问题类型，填写标题和详细描述
3. 提交后等待客服回复
4. 可在工单列表中查看处理进度

---

## 消息通知

EzPanel 通过以下渠道推送通知：

- **站内消息**：登录后在通知铃铛图标查看
- **邮件**：注册邮箱接收（需管理员配置 SMTP）
- **Telegram**：绑定 Telegram 后接收（需管理员配置 Bot）

### 绑定 Telegram

1. 在 Telegram 中找到站点 Bot
2. 发送 `/bindaccount`
3. 按提示在面板中获取验证码
4. 发送 `/bindcode <验证码>` 完成绑定

## 个人设置

进入右上角头像 → 「个人设置」：

| 设置项 | 说明 |
|--------|------|
| 邮箱 | 修改登录邮箱（需验证） |
| 密码 | 修改登录密码 |
| 两步验证 | 开启 / 关闭 2FA |
| 设备管理 | 查看和登出其他设备的登录会话 |
| Telegram 绑定 | 绑定 / 解绑 Telegram 账号 |
