# Руководство пользователя

**Язык**: [English](./04_user-guide.md) | [简体中文](./04_user-guide_CN.md) | [繁體中文](./04_user-guide_TW.md) | Русский | [فارسی](./04_user-guide_FA.md)

Настоящий документ предназначен для рядовых пользователей EzPanel и описывает порядок регистрации, входа в систему, получения подписки, приобретения тарифных планов и другие функции.

---

## Регистрация и вход в систему

### Регистрация аккаунта

1. Откройте адрес панели и нажмите «Регистрация»
2. Введите адрес электронной почты и пароль
3. Если на сайте включена верификация по электронной почте, проверьте почту и перейдите по ссылке подтверждения
4. Если сайт требует инвайт-код, введите его и отправьте форму

### Вход в систему

Введите адрес электронной почты и пароль, использованные при регистрации. Отметьте «Запомнить меня» для продления срока действия сессии.

### Забыли пароль

1. На странице входа нажмите «Забыли пароль?»
2. Введите адрес электронной почты, указанный при регистрации — система отправит письмо для сброса пароля
3. Перейдите по ссылке в письме и установите новый пароль

## Двухфакторная аутентификация (2FA)

Включение двухфакторной аутентификации эффективно защищает аккаунт от несанкционированного доступа.

### Включение 2FA

1. Перейдите в «Настройки профиля» → «Настройки безопасности»
2. Нажмите «Включить двухфакторную аутентификацию»
3. Отсканируйте QR-код с помощью Google Authenticator, Aegis или аналогичного приложения
4. Введите 6-значный код из приложения для подтверждения

### Использование 2FA при входе

После ввода адреса электронной почты и пароля система запрашивает 6-значный динамический код подтверждения.

> **Важно**: Обязательно сохраните резервные коды восстановления — при их утере доступ к аккаунту восстановить невозможно.

## Подписка

Ссылка на подписку используется для импорта информации об узлах в клиентские приложения (Clash, Sing-box и другие).

### Получение ссылки на подписку

1. После входа перейдите в раздел «Моя подписка» или на главную страницу
2. Скопируйте ссылку на подписку
3. Вставьте её в поле «Добавить подписку» в клиентском приложении и обновите

### Форматы подписки

EzPanel поддерживает несколько форматов подписки. Распространённые клиенты и рекомендуемые форматы:

| Клиент | Рекомендуемый формат |
|--------|----------------------|
| Clash / Mihomo | Формат Clash |
| Sing-box | Формат Sing-box |
| v2rayN / Nekoray | Универсальный Base64 |
| Quantumult X | Универсальный формат |

> Конкретные параметры формата могут отличаться в зависимости от настроек сайта. Уточните у администратора.

### Обновление подписки

Содержимое подписки обновляется по мере изменения узлов. Рекомендуется включить автоматическое периодическое обновление в клиентском приложении (например, каждый час или раз в день).

## Приобретение тарифного плана (функция версии Pro)

> Следующие функции доступны только при наличии установленной профессиональной лицензии на сайте.

### Покупка тарифного плана

1. Перейдите на страницу «Тарифы» и просмотрите доступные варианты
2. Выберите тариф и период оплаты (ежемесячно / ежеквартально / ежегодно и т. д.)
3. Подтвердите заказ и выберите способ оплаты
4. После оплаты тарифный план вступает в силу немедленно

### Продление подписки

До истечения срока действия тарифного плана перейдите в «Мою подписку» для досрочного продления — срок действия будет продлён начиная с текущей даты истечения.

### Что делать при исчерпании трафика

После исчерпания трафика подписка приостанавливается. Варианты действий:
- Приобрести дополнительный пакет трафика (если доступно)
- Дождаться автоматического сброса в начале следующего расчётного периода
- Вручную продлить подписку для сброса трафика

## Активация кода

Некоторые сайты поддерживают активацию тарифного плана или пополнение баланса с помощью кода.

1. Перейдите на страницу «Активация кода» или «Активировать»
2. Введите код активации
3. Нажмите «Активировать» — баланс или тарифный план будут зачислены немедленно после успеха

## Приглашение друзей

1. Перейдите на страницу «Реферальная программа» или «Пригласить»
2. Скопируйте персональную реферальную ссылку или инвайт-код
3. Поделитесь с друзьями — при их регистрации и оплате вы получите комиссию

### Просмотр комиссионных

На странице «Реферальная программа» доступна следующая информация:
- Количество приглашённых пользователей
- Накопленная сумма комиссионных
- Доступный баланс для вывода

### Запрос на вывод средств (версия Pro)

При достижении минимальной суммы вывода подайте заявку на странице «Вывод средств» — после проверки администратором средства будут перечислены.

## Обращения в поддержку (функция версии Pro)

При возникновении проблем обратитесь в службу поддержки сайта:

1. Перейдите в «Тикеты» → «Создать тикет»
2. Выберите тип проблемы, введите заголовок и подробное описание
3. Отправьте и дождитесь ответа специалиста поддержки
4. Статус обработки можно отслеживать в списке тикетов

## Уведомления

EzPanel отправляет уведомления по следующим каналам:

- **Внутрисистемные сообщения**: доступны после входа в систему через значок колокольчика
- **Электронная почта**: отправляется на адрес, указанный при регистрации (требует настройки SMTP администратором)
- **Telegram**: после привязки Telegram (требует настройки бота администратором)

### Привязка Telegram

1. Найдите бота сайта в Telegram
2. Отправьте команду `/bindaccount`
3. Следуйте инструкциям — получите код подтверждения в панели
4. Отправьте команду `/bindcode <код>` для завершения привязки

## Настройки профиля

Перейдите к аватару в правом верхнем углу → «Настройки профиля»:

| Параметр | Описание |
|----------|----------|
| Электронная почта | Изменить адрес для входа (требует верификации) |
| Пароль | Изменить пароль для входа |
| Двухфакторная аутентификация | Включить / отключить 2FA |
| Управление устройствами | Просмотр и завершение сессий на других устройствах |
| Привязка Telegram | Привязать / отвязать аккаунт Telegram |
