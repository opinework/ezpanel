# 使用者手冊

**語言**: [English](./04_user-guide.md) | [简体中文](./04_user-guide_CN.md) | 繁體中文 | [Русский](./04_user-guide_RU.md) | [فارسی](./04_user-guide_FA.md)

本文件面向 EzPanel 的一般使用者，介紹註冊、登入、訂閱取得、方案購買等功能的使用方法。

---

## 註冊與登入

### 註冊帳號

1. 開啟面板位址，點擊「註冊」
2. 填寫電子郵件和密碼
3. 如果站台開啟了電子郵件驗證，請查收郵件並點擊驗證連結
4. 如果站台要求邀請碼，填寫邀請碼後送出

### 登入

輸入註冊時使用的電子郵件和密碼登入。勾選「記住我」可延長登入有效期。

### 忘記密碼

1. 在登入頁點擊「忘記密碼」
2. 輸入註冊電子郵件，系統會發送重設郵件
3. 點擊郵件中的連結設定新密碼

---

## 兩步驗證（2FA）

開啟兩步驗證可有效防止帳號被盜。

### 開啟 2FA

1. 進入「個人設定」→「安全設定」
2. 點擊「開啟兩步驗證」
3. 用 Google Authenticator 或 Aegis 等應用程式掃描二維碼
4. 輸入 App 中顯示的 6 位驗證碼確認開啟

### 登入時使用 2FA

登入時輸入電子郵件密碼後，會要求輸入 6 位動態驗證碼。

> **提示**：請務必儲存好備用恢復碼，遺失後無法找回。

## 訂閱

訂閱連結用於將節點資訊匯入到客戶端（如 Clash、Sing-box 等）。

### 取得訂閱連結

1. 登入後進入「我的訂閱」或首頁
2. 複製訂閱連結
3. 貼上到客戶端的「新增訂閱」處並更新

### 訂閱格式

EzPanel 支援多種訂閱格式，常見客戶端對應格式：

| 客戶端 | 推薦格式 |
|--------|----------|
| Clash / Mihomo | Clash 格式 |
| Sing-box | Sing-box 格式 |
| v2rayN / Nekoray | 通用 Base64 |
| Quantumult X | 通用格式 |

> 具體格式參數可能因站台設定而有所不同，請聯繫站台管理員確認。

### 訂閱刷新

訂閱內容會隨節點變化而更新。建議在客戶端中開啟定時自動更新（如每小時或每天一次）。

## 方案購買（Pro 版功能）

> 以下功能需要站台安裝專業版授權後才可使用。

### 購買方案

1. 進入「方案」頁面，瀏覽可用方案
2. 選擇方案和計費週期（月付 / 季付 / 年付等）
3. 確認訂單，選擇付款方式
4. 完成付款後方案立即生效

### 續費

方案到期前可進入「我的訂閱」提前續費，續費從當前到期時間順延。

### 流量用完怎麼辦

流量耗盡後訂閱將暫停，可選擇：
- 購買流量包補充（如支援）
- 等待下個計費週期自動重設
- 手動續費觸發流量重設

---

## 序號兌換

部分站台支援透過序號兌換方案或餘額。

1. 進入「序號兌換」或「兌換」頁面
2. 輸入序號代碼
3. 點擊「兌換」，成功後餘額 / 方案立即到帳

---

## 邀請好友

1. 進入「邀請」或「推廣」頁面
2. 複製專屬邀請連結或邀請碼
3. 分享給好友，好友註冊並消費後你會獲得佣金

### 查看佣金

在「推廣 / 邀請」頁面可查看：
- 已邀請使用者數
- 累計佣金金額
- 可提現餘額

### 申請提現（Pro 版）

滿足最低提現金額後，在「提現」頁面發起申請，管理員審核後付款。

---

## 工單（客服）（Pro 版功能）

如遇問題需要聯繫站台客服：

1. 進入「工單」→「新建工單」
2. 選擇問題類型，填寫標題和詳細說明
3. 送出後等待客服回覆
4. 可在工單清單中查看處理進度

---

## 訊息通知

EzPanel 透過以下管道推送通知：

- **站內訊息**：登入後在通知鈴鐺圖示查看
- **電子郵件**：註冊電子郵件接收（需管理員設定 SMTP）
- **Telegram**：綁定 Telegram 後接收（需管理員設定 Bot）

### 綁定 Telegram

1. 在 Telegram 中找到站台 Bot
2. 發送 `/bindaccount`
3. 按提示在面板中取得驗證碼
4. 發送 `/bindcode <驗證碼>` 完成綁定

---

## 個人設定

進入右上角頭像 → 「個人設定」：

| 設定項 | 說明 |
|--------|------|
| 電子郵件 | 修改登入電子郵件（需驗證） |
| 密碼 | 修改登入密碼 |
| 兩步驗證 | 開啟 / 關閉 2FA |
| 裝置管理 | 查看和登出其他裝置的登入工作階段 |
| Telegram 綁定 | 綁定 / 解除綁定 Telegram 帳號 |
