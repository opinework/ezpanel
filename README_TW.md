**語言**: [English](./README.md) | [简体中文](./README_CN.md) | 繁體中文 | [Русский](./README_RU.md) | [فارسی](./README_FA.md)

## 版本說明

### Community（社群版）— 永久免費

社群版包含個人使用者的基礎功能，滿足一般使用者的正常使用。**永久免費，不設期限。**

- 使用者管理、訂閱管理
- 節點管理（XrayR/XrayM 對接）
- 方案管理
- 系統管理

> 社群版為閉源軟體，官方不提供技術支援和客製化需求。

### Pro（專業版）— 商業授權

在社群版基礎上增加：

- 訂單 / 發票 / 行銷管理
- 付款閘道整合（支付寶、微信、Stripe 等）
- 工單系統
- 知識庫
- 佣金 / 推廣系統
- 財務報表與資料分析

### Enterprise（企業版）— 商業授權

在專業版基礎上增加：

- SSH 節點遠端管理
- 應用程式內購（IAP）支援

> 購買 Pro 或 Enterprise 授權請至 [https://opine.work](https://opine.work)。
