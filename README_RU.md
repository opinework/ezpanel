## Описание версий

### Community — Бесплатно навсегда

Версия Community включает базовые функции для индивидуальных пользователей. **Бесплатно навсегда, без ограничений по сроку.**

- Управление пользователями и подписками
- Управление узлами (подключение XrayR/XrayM)
- Управление тарифными планами
- Системное администрирование

> Версия Community является программным обеспечением с закрытым исходным кодом; официальная техническая поддержка и кастомизация не предоставляются. Для получения доступа к исходному коду необходимо приобрести коммерческую лицензию.

### Pro — Коммерческая лицензия

Дополнительно к версии Community:

- Управление заказами / счетами / маркетингом
- Интеграция платёжных шлюзов (Alipay, WeChat Pay, Stripe и др.)
- Система тикетов
- База знаний
- Реферальная / партнёрская программа
- Финансовая отчётность и анализ данных
- **Доступ к репозиторию исходного кода** (на срок действия лицензии)

### Enterprise — Коммерческая лицензия

Дополнительно к версии Pro:

- Удалённое управление узлами через SSH
- Поддержка внутренних покупок приложений (IAP)

> Приобрести лицензию Pro или Enterprise: [https://opine.work](https://opine.work).

## Политика выпуска

- **Закрытый исходный код**: EzPanel — проприетарное программное обеспечение с закрытым исходным кодом, который не находится в открытом доступе; для получения доступа к репозиторию необходимо приобрести коммерческую лицензию.
- **Бесплатно навсегда**: версия Community доступна для постоянного бесплатного использования без ограничений функций.
- **Доступ к исходному коду**: если вам необходим доступ к исходному коду, приобретите коммерческую лицензию (Pro или Enterprise). Обладатели коммерческой лицензии получают доступ к репозиторию на весь срок действия лицензии.
- **Периодические обновления**: новые версии выпускаются регулярно с исправлениями и улучшениями.
- **Анонсы**: подпишитесь на [@OpineWorkPublish](https://t.me/OpineWorkPublish) в Telegram.
